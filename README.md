# Message-App
A mini messaging system in Python demonstrating OOP, user interaction, and message handling logic

►Python Messaging App
This is a simple messaging system built in Python using Object-Oriented Programming (OOP).
It simulates how users send, receive, read, and manage messages — along with features like blocking users and message status tracking.

►Features
•Send messages between users
•Read messages with detailed view
•Delete messages
•Block users
•Message status (Read / Unread)
•Timestamp for messages

►Concepts Used
•Classes and Objects
•Object Interaction (User ↔ Message)
•Encapsulation (data inside objects)
•Lists for storing messages
•Conditional logic

►Future Improvements
•Unblock users feature
•Message filtering (read / unread / replied)
•Store messages in a database (SQLite)
•Backend integration using Flask / FastAPI
•Simple frontend (HTML/CSS/JS)

►About This Project
This is my first self-built Python project where I applied OOP concepts to create a working system instead of just solving problems.
